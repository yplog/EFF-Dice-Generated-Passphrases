# EFF Dice-Generated Passphrases

Please read the [article](https://www.eff.org/dice) before you start.

## Usage:  
```bash
main.py [GENERATOR] [MODE] 
```   
## Generator:  
For more information about generators: https://www.eff.org/dice  
* -l  --longwordlist   
* -s1 --shortwordlist1  
* -s2 --shortwordlist2  
#### Example:  
```bash
main.py -l 
``` 
## Mode:  
Default mode is offline.  
* --online  
* --offline  
#### Example:  
```bash
main.py -l --online
```
